# Установка кнопки обращения в Bitrix24

После деплоя проекта создайте в админском Bitrix24 локальное серверное приложение с пользовательским интерфейсом.

## URL приложения

- URL приложения: `https://b24support.abs7.ru:8443/bitrix/app`
- Путь для первоначальной установки: `https://b24support.abs7.ru:8443/bitrix/app/install`

## Права приложения

Выберите права, соответствующие областям:

- `placement` — встройки приложений;
- `im` — мессенджер;
- `user` — сведения о текущем пользователе.

Публиковать приложение в Маркетплейсе не требуется.

## Установка

1. Откройте добавленное локальное приложение под администратором Bitrix24.
2. Нажмите **«Установить / обновить кнопку»**.
3. Установщик зарегистрирует `IM_TEXTAREA` с обработчиком `/bitrix/app/widget` и вызовет `BX24.installFinish()`.
4. Обновите страницу Bitrix24 и откройте групповой чат обращения.
5. Над полем ввода появится пункт **«Обращение»**.

Внутри пункта:

- для открытого обращения показывается **«Обращение закрыто»**;
- после закрытия показывается **«Удалить чат»**;
- в постороннем групповом чате будет сообщение, что чат не связан с обращением.

## Ограничение Bitrix24

Встройка `IM_TEXTAREA` регистрируется для типа чатов, а не для конкретных `dialogId`. Поэтому внешний пункт **«Обращение»** отображается во всех групповых чатах. Backend разрешает действия только в чатах, связанных с `support_ticket`, и только сотрудникам, выбранным в админке как специалисты техподдержки.
